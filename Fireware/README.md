# 机械臂硬件控制项目

## 1. 项目简介

本项目基于 **STM32F107（Cortex-M3 内核）微控制器**，使用 **Keil uVision（ARMCC）** 作为开发与编译环境，实现一套面向科研与工程验证的 **手动控制（Manual Operation）模式嵌入式系统**。

STM32F107 属于 **Connectivity Line**，片上集成 **CAN / USB OTG / Ethernet / 多路 USART / DMA** 等外设，适合用于：

* 🧪 控制系统与通信协议科研实验
* 🔧 多外设协同控制与实时性验证
* 📚 STM32F1 Connectivity Line 架构学习与工程复现

本仓库已按 **科研级工程规范** 进行整理，适合作为：

* 硕士阶段实验工程仓库
* STM32F107 + Keil 项目模板

## 4. 启动文件与系统配置（关键）

### 4.1 启动文件
本工程使用 **STM32F107 专用启动文件**，位于：

```text
CMSIS/DeviceSupport/startup/startup_stm32f10x_cl.s
```

**请确认**：
	* 启动文件为 **Connectivity Line（cl）** 版本
	* 与 MCU 型号（Flash/RAM 大小）匹配

### 4.2 系统时钟配置
	* 外部晶振：8 MHz（常见）
	* 系统主频：72 MHz
	* 时钟初始化位于：
		System/system_stm32f10x.c

	> ⚠️ 修改外部晶振或 PLL 参数时，请同步检查 CAN / USB / Ethernet 时钟配置。

## 6. Git 版本管理规范

### 6.1 忽略规则说明

已通过  .gitignore忽略：
* Keil 用户文件：`*.uvoptx` / `*.uvguix.*`
* 编译产物：`Objects/` / `Listings/` / `*.axf` / `*.hex`
* 调试配置：`*.dbgconf`

### 6.2 提交规范（推荐）

```text
feat: 新增外设或功能模块
fix: 修复硬件/驱动问题
exp: 实验性修改
refactor: 结构重构
cleanup: 工程整理（无功能改动）
```

示例：

```bash
git commit -m "feat: add STM32F107 CAN manual control"
```
